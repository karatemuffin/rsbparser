Hilfe
  1. Öffne Sokrates
  2. Selektiere das gewünschte Schuljahr
  3. Klicke auf den Menüpunkt Beurteilung
  4. Wähle was zu Drucken gewünscht (z.B. Mahnungen)
  5. Fülle das Suchraster
  6. Auswahl der gewünschten Schüler
  7. Bestätige mit "Drucken"
  8. Das nun gespeicherte PDF auf dieser Seite hochladen.
  1. Bereite Drucker vor
    1. Die Seitenlade des Druckers öffnen
    2. Die RSb Formulare einlegen
    3. Das Papierformat auf A5 auswähen
  2. Druckeinstellungen
    1. Setze die Druckgröße auf "Tatsächliche Größe"
    2. Deaktiviere das beidseitige Bedrucken
    3. Unter Eigenschaften -> Grundeinstellungen: stelle die Papiergröße auf A5
    4. Unter Eigenschaften -> Papierzufuhr: wähle das Mehrzweckfach
